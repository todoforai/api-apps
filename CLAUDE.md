ElevenLabs CLI available globally: `elevenlabs`

## elevenlabs-api
Raw API wrapper CLI. Run with `bun run elevenlabs-api/src/index.ts`.

API reference: https://elevenlabs.io/docs/api-reference/text-to-speech/convert/llms.txt

## suno-api
Suno music generation CLI. Run with `bun run suno-api/src/index.ts`.

API reference: https://docs.sunoapi.org/llms.txt
